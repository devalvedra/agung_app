# Assets Directory

## Required Images

Place the following image files in this directory:

### truck_icon.png

A delivery truck icon for the tracking map marker.

**Specifications:**

- Format: PNG with transparent background
- Recommended size: 96x96 or 128x128 pixels
- Color: Blue or any color that contrasts well with the map

**Where to get:**

1. **Free Icons:**

   - Flaticon: https://www.flaticon.com/free-icons/delivery-truck
   - Icons8: https://icons8.com/icons/set/delivery-truck
   - Freepik: https://www.freepik.com/search?format=search&query=delivery%20truck%20icon

2. **Create Custom:**

   - Use Figma, Canva, or any graphic design tool
   - Create a simple truck silhouette
   - Export as PNG

3. **Fallback:**
   - If not provided, the app will use a default blue map marker

**Note:** After adding the icon, run `flutter pub get` to ensure assets are registered.
